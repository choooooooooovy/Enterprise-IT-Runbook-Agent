# RB-003 비밀번호 재설정 Runbook

## Category
Password

## Symptoms
- 비밀번호를 잊었거나 만료되어 로그인할 수 없다.
- 비밀번호 변경 후 일부 서비스에서 이전 자격 증명을 요구한다.
- 반복 실패로 계정이 잠겼다.

## Required Information
- 대상 계정과 영향 서비스
- 사용자 본인 확인 결과
- 계정 잠금/만료 상태와 마지막 성공 로그인 시각
- 비밀번호 변경 직후인지 여부
- 회사 네트워크 또는 셀프서비스 접근 가능 여부

## Resolution Steps
1. 승인된 절차로 사용자 신원을 확인한다.
2. 계정 잠금과 비밀번호 만료 상태를 확인한다.
3. 가능하면 셀프서비스 비밀번호 재설정을 안내한다.
4. 관리자 재설정이 필요하면 임시 비밀번호와 변경 요구 정책을 적용한다.
5. 저장된 이전 자격 증명과 모바일/메일 클라이언트 세션을 갱신한다.
6. 비밀번호를 공유하거나 ticket 본문에 기록하지 않도록 안내한다.

## Verification
- 새 비밀번호로 기본 로그인과 핵심 서비스 접근이 성공하는지 확인한다.
- 계정 잠금이 다시 발생하지 않는지 확인한다.

## Escalation Conditions
- 본인 확인에 실패하거나 계정 탈취가 의심된다.
- 재설정 후에도 여러 서비스에서 인증 동기화가 장시간 실패한다.
- 관리자 또는 서비스 계정의 긴급 변경 요청이다.
